Propriedades Position e Display no CSS

Conceito position

A propriedade position do CSS define como um elemento será posicionado dentro da página e em relação a outros elementos.

Ela permite controlar a posição de um elemento utilizando propriedades como:

  • top
  • bottom
  • left
  • right

Dependendo do valor utilizado, o elemento pode seguir o fluxo normal da página ou ser retirado dele, permitindo posicionamentos mais específicos.

Os principais valores da propriedade position são:

  • static
  • relative
  • absolute
  • fixed
  • fixed

position: static

O valor static é o valor padrão de todos os elementos HTML.

Quando um elemento possui position: static, ele segue o fluxo normal da página, aparecendo exatamente na ordem em que está no código HTML.

Nesse modo, o elemento não pode ser movido usando:

  • top
  • bottom
  • left
  • right

Ou seja, essas propriedades não têm efeito quando o position é static.

Esse valor geralmente é utilizado quando não é necessário alterar o posicionamento do elemento.

Elemento Static

position: relative

O valor relative permite reposicionar um elemento em relação à sua posição original.

Diferente do static, ele aceita as propriedades:

  • top
  • bottom
  • left
  • right

Mesmo sendo movido, o elemento continua ocupando seu espaço original no layout. Ou seja, os outros elementos continuam se comportando como se ele ainda estivesse na posição inicial.

Outra característica importante é que elementos com position: relative podem servir como referência para elementos com position: absolute.

Elemento Relative

position: absolute

O valor absolute remove o elemento do fluxo normal da página.

Isso significa que os outros elementos da página se comportam como se ele não existisse.

Um elemento com position: absolute é posicionado em relação ao elemento pai mais próximo que tenha position diferente de static.

Se não existir nenhum elemento pai com position definido, ele será posicionado em relação ao body da página.

Esse tipo de posicionamento é muito utilizado para:

  • menus suspensos
  • pop-ups
  • elementos sobrepostos
  • ícones posicionados sobre imagens
Elemento Absolute

position: fixed

O valor fixed fixa o elemento em uma posição específica da tela, independente da rolagem da página.

Isso significa que o elemento permanece sempre visível, mesmo quando o usuário faz scroll.

Assim como o absolute, ele também é removido do fluxo normal da página.

Esse valor é muito utilizado para:

  • barras de navegação fixas
  • botões de voltar ao topo
  • menus flutuantes
  • chat de suporte

Um exemplo seria o header.

position: sticky

O valor sticky combina características do relative e do fixed.

Inicialmente, o elemento se comporta como relative, seguindo o fluxo normal da página.

Porém, quando o usuário rola a página e o elemento atinge um determinado ponto definido por top, bottom, left ou right, ele passa a se comportar como fixed, ficando fixo na tela.

Esse valor é muito utilizado em:

  • menus de navegação
  • cabeçalhos de tabelas
  • menus laterais
  • chat de suporte

Display: Block

O Display Block ele ocupa toda a linha disponível, mesmo que não tenha conteúdo suficiente para preencher a linha.

Exemplo: Um parágrafo de texto — ocupa a linha inteira independente do tamanho.
