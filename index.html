Propriedades Position e Display no CSS

Conceito position

A propriedade position do CSS define como um elemento será posicionado dentro da página e em relação a outros elementos.

Ela permite controlar a posição de um elemento utilizando propriedades como:

  • top
  • bottom
  • left
  • right

Dependendo do valor utilizado, o elemento pode seguir o fluxo normal da página ou ser retirado dele, permitindo posicionamentos mais específicos.

Os principais valores da propriedade position são:

  • static
  • relative
  • absolute
  • fixed
  • fixed

position: static

O valor static é o valor padrão de todos os elementos HTML.

Quando um elemento possui position: static, ele segue o fluxo normal da página, aparecendo exatamente na ordem em que está no código HTML.

Nesse modo, o elemento não pode ser movido usando:

  • top
  • bottom
  • left
  • right

Ou seja, essas propriedades não têm efeito quando o position é static.

Esse valor geralmente é utilizado quando não é necessário alterar o posicionamento do elemento.
